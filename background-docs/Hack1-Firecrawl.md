## Challenge

Combine Firecrawl Search with ElevenAgents.

Build something with both Firecrawl Search and ElevenAgents, then submit a high-quality viral-style video demonstrating what you've built.

## What to Build

Build something that uses **both**:
- [Firecrawl](https://firecrawl.dev/) – turns any website into clean, LLM-ready data. The Search API lets you search the web and get structured content back in a single call, ideal for giving AI agents real-time knowledge about anything on the internet.
- [ElevenAgents](https://elevenlabs.io/agents) – ElevenLabs' conversational agents platform. Build and deploy natural, human-sounding voice agents that access external tools and take actions on behalf of users.

Then create and submit a **high-quality viral-style demo video** of your project.
